# Дастури истифода ва деплой

## 1. Гирифтани токени бот
1. Дар Telegram @BotFather-ро ёбед
2. `/newbot` фиристед ва номи ботро диҳед
3. Токенеро, ки ба шумо медиҳад, нигоҳ доред (масалан: `123456:ABC-DEF...`)

## 2. Ёфтани ADMIN_ID (ID-и телеграми худатон)
1. Дар Telegram @userinfobot-ро ёбед
2. `/start` фиристед
3. Он ID-и шуморо мефиристад (масалан: `123456789`) — инро нигоҳ доред

## 3. Гузоштан ба GitHub
```bash
cd kfs_bot
git init
git add .
git commit -m "Telegram bot for food ordering"
git remote add origin https://github.com/USERNAME/REPO.git
git push -u origin main
```

## 4. Деплой дар Render.com (ройгон)
1. Ба [render.com](https://render.com) ворид шавед
2. "New" → "Web Service"
3. Репозиторияи GitHub-и худро интихоб кунед
4. Танзимот:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `python bot.py`
5. Дар қисми **Environment Variables** ду тағйирёбанда илова кунед:
   - `BOT_TOKEN` = токени боти шумо
   - `ADMIN_ID` = ID-и телеграми шумо
6. "Create Web Service"-ро пахш кунед

## 5. Нигоҳ доштани бот доимо фаъол
Плани ройгони Render баъд аз 15 дақиқаи бетаваҷҷуҳӣ бот метавонад хоб равад.
Барои пешгирӣ:
1. Ба [uptimerobot.com](https://uptimerobot.com) сабти ном кунед (ройгон)
2. Монитори HTTP созед, ки ҳар 5 дақиқа ба линки Render-и шумо (масалан `https://ном-и-бот.onrender.com`) дархост мефиристад

## Тарзи кор
### Барои мизоҷон:
- `/start` — оғози кор, номро мегирад
- 🍗 Хӯрокҳо — дидани менюи хӯрок ва иловаи ба сават
- 🛒 Саватам — дидани сават, тасдиқи фармоиш (пардохт нақд)
- 📦 Фармоишҳои ман — таърихи фармоишҳо

### Барои admin (танҳо шумо):
- ⚙️ Панели admin пайдо мешавад дар менюи асосии шумо
- ➕ Иловаи хӯрок — номи хӯрок ва нархро дохил мекунед
- 📋 Рӯйхати хӯрокҳо — дидани ҳамаи хӯрокҳо
- 📊 Омори моҳона — хӯрокҳои бисёр фурӯхташуда дар 30 рӯзи охир

## Тахфиф
Агар маблағи фармоиш ≥ 1000 сомонӣ бошад, ба таври худкор 10% тахфиф дода мешавад.
Инро дар bot.py тавассути `DISCOUNT_THRESHOLD` ва `DISCOUNT_PERCENT` тағйир дода метавонед.
